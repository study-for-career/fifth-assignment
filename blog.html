Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents (html). It represents the document as nodes and objects. So that a programming language can access the documents.

How do you select an element from the DOM?

There are several ways to select an element. Most used selector: byID, byClassName, byTagName, querySelector, querySelectorAll

What is event delegation in the context of the DOM, and why is it useful?

Instead of working on an element by targeting it, working on the element through its parent is called event delegation.
By working through event delegation we can not only operate on the current element but also access any new element added to the same parent. Which helps us to do our work easier, writing less code and faster.

How do you manipulate an element's attributes and styles using the DOM?

Manipulate an attributes: we can use getAttribute() to get an element's attribute. setAttribute() to add a new attribute. Also has removeAttribute();
To give style an element we can use elementName.style.propertyName = 'value'
Also we can create style element and add to head section to give our desired style using js.
